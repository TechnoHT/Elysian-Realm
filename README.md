### Setup

Install dependencies:
```
npm install
```

Start webpack server:
```
npm run serve
```

Watch file changes:
```
npm run watch
```
When adding new files, this command needs to be restarted.

### Other Scripts

Test the guide file:
```
npm test
```

Deploy to GitHub Pages:
```
npm run deploy
```
